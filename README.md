# node-callml
Sends applications to CallML 6.1.1 service

## Running the tests

Create a `.callmlrc` file in project root directory. Add your CallML test credentials to it. e.g.

```js
{
  "url": "https://www.callcreditsecure.co.uk/Services/CallML/CallML6.asmx",
  "company": "[company]",
  "username": "[username]",
  "password": "[password]"
}
```
